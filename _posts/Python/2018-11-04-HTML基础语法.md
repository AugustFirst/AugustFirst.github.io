# HTML
* HTML:超文本标记语言---HyperText Markup Language
* 基本结构
	``` 
	<!DOCTYPE html>
	<html lang="en">
	<head>
	    <meta charset="UTF-8">
	    <title>Title</title>
	</head>
	<body>

	</body>
	</html>

	```
	
* `body`中的标签分类
> 块级元素
> 行内元素
> 单标签
> 双标签

* 常见标签

1.和文本相关
> 标题h1-h6

> 段落`<p></p>`

> 加粗 `<strong></strong>  <b></b>`

> 倾斜`<em></em> <i></i>`

> 下划线`<ins></ins> <u></u>`

> 删除线`<del></del> <s> </s>`

2.通用容器
> `<div> </div>`
`<span></span>`

3.其他标签
> 换行`<br/>`

> 水平线`<hr/>`

> 空格 `&nbsp;`

> 小于 `&lt`

> 大于 `&gt`

4.超链接
> `<a></a> href target =_blank / _self`

> 图片超链接`<a><img/></a>`

5.图片
> `<img src="" /> title alt width height`
